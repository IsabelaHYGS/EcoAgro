# EcoAgro

# 🌿 Projeto EcoAgro - Agro é forte: futuro sustentável

Esse site foi desenvolvido com a ideia principal de mostrar como o agronegócio pode continuar produzindo muito, mas sem destruir o meio ambiente, focando no equilíbrio e na sustentabilidade do campo.

## 💡 O problema e a solução
Na escola e nos jornais, sempre vemos a discussão sobre como o agro precisa produzir alimentos para o mundo, mas muitas vezes prejudicando o solo ou gastando água demais. Para tentar ajudar a resolver isso, foi desenvolvido uma calculadora de eficiência sustentável. O produtor rural (ou qualquer pessoa) entra no site, coloca o tamanho da terra (hectares) e o quanto colheu (toneladas). O site calcula a produtividade por hectare na hora e dá uma nota de sustentabilidade baseada no tipo de manejo que ele usa (se é o convencional, integrado ou agroecologia).

## 🛠️ Tecnologias usadas

HTML: Para montar toda a estrutura do site (textos, formulário, seções e rodapé).

CSS: Para deixar o site bonito, usamos tons de verde e marrom terra para combinar com o tema. Também configuramos o layout (com Flexbox e Grid) para que o site funcione direitinho tanto no computador quanto na tela do celular (responsivo).

JavaScript: É a "inteligência" do site. Ele pega os números que o usuário digita, faz as contas de divisão para descobrir a produtividade, decide qual mensagem de feedback mostrar e até muda a cor do resultado (verde para bom, vermelho para alerta) sem precisar atualizar a página.

## 📁 Como o projeto está organizado
index.html — Todo o conteúdo e o formulário do simulador.

style.css — O visual, as cores e a formatação para celular.

script.js — Os cálculos da nossa calculadora e a lógica do site.

Nota: As imagens do site estão puxando links direto da internet (Unsplash), então precisa estar conectado para elas carregarem.

## 🚀 Como testar no computador
1. Baixe os arquivos e coloque todos eles juntos dentro de uma mesma pasta.
2. Dê dois cliques no arquivo index.html. Ele vai abrir direto no seu navegador de internet e já dá para usar o simulador.
